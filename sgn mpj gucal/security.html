Security for Gas Usage

Do's

  • Keep the regulator off when not in use


  • While going for work, function or some other work please check the gas is off or not


  • It is necessary to check it daily night


  • Please check the gas leakage


  • If any gas pipe leakage or regulator leakage ,Dont be panic # Open the window #


Dont's

  1. Don’t keep the gas burning without any vessel


  2. Avoid using high flame for all vessels and the recipes


  3. If any gas leakage don’t switch on/ off the lights or any electrical appliances
